\doxysubsection{Detailed Description}
Dialog window for displaying real-\/time pressure and temperature graphs. 

This class handles data visualization using a \doxylink{class_graph_widget}{Graph\+Widget}, and receives real-\/time updates via \doxylink{class_web_socket_client}{Web\+Socket\+Client}. It also emits signals on show/hide events and when a specific flag is sent.
